# BookWright — AI小说写作与续写器

AI 驱动的智能小说写作工具。FastAPI 后端 + React 前端，每部小说独立 SQLite 存档。

## Agent skills

### Issue tracker

Issues live as GitHub issues on the repo. See `docs/agents/issue-tracker.md`.

### Triage labels

Default English labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context — one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
